# FFGLS Tournament 2026 - Website

Situs statis sederhana untuk sistem informasi turnamen Free Fire (48 tim) dengan layout putih-pink, tema clean.

Cara menjalankan:

1. Buka file `index.html` di browser (cukup double-click atau `Open File`).
2. Untuk pengembangan lokal, jalankan server statis (opsional):

```powershell
# dengan Python 3
python -m http.server 8000
# lalu buka http://localhost:8000
```

File utama:

- `index.html` — halaman utama berisi semua informasi turnamen
- `css/style.css` — gaya tampilan

Konten dapat disesuaikan: nama tim, jadwal, kontak, dan pembagian hadiah.
